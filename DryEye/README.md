# 干眼症监测系统123456

本系统通过采集眼睑信号并分析眨眼特征，实现对用户干眼风险的实时监测与评估。

---

## 一、项目背景与核心目标

干眼症是现代生活中常见的眼科疾病。本项目通过监测眨眼频率、眼部稳定性等关键指标，帮助用户及时了解眼部健康状况，预防干眼症的发生。

系统核心目标是通过可穿戴硬件采集眼睑电压信号，利用后端数字信号处理算法进行分析，最终在前端大屏上实时呈现干眼风险状态和预警信息。
