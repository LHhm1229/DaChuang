# 干眼症监测系统 (Dry Eye Monitoring System)

本项目是一个基于 Web 技术的干眼症实时监测系统界面，旨在通过传感器数据实时分析用户的眼部状态，评估干眼风险并提供预警。

## 项目背景
干眼症是现代生活中常见的眼科疾病。本项目通过监测眨眼频率、眼部稳定性等关键指标，帮助用户及时了解眼部健康状况，预防干眼症的发生。

## 主要功能
- **实时监控**: 实时显示眼部状态、健康评分、眨眼频率等数据。
- **数据分析**: 提供眼部健康趋势、眨眼频率趋势、稳定性变化等可视化图表。
- **蓝牙连接**: 支持通过 Web Bluetooth API 连接外部监测设备。
- **WebSocket 同步**: 支持通过 WebSocket 接收远程传感器数据。
- **系统设置**: 可自定义检测灵敏度、干眼阈值及警报配置。

## 技术栈
- **前端框架**: React + TypeScript
- **UI 组件**: Tailwind CSS + Radix UI (Shadcn UI)
- **图表库**: Recharts
- **图标**: Lucide React
- **通信**: Web Bluetooth API, WebSocket

## 快速开始

1. **安装依赖**:
   ```bash
   npm install
   ```

2. **启动开发服务器**:
   ```bash
   npm run dev
   ```

3. **访问应用**:
   打开浏览器访问 `http://localhost:5173` (或终端显示的端口)。
