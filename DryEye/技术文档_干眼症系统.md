**关键参数:**
- 幅度阈值 0.10: 低于此值的波形不认为是眨眼
- 最小时长 0.04s: 捕捉微眨眼
- 最大时长 1.0s: 过滤异常长信号
- 不应期 0.15s: 两次眨眼最小间隔

### 5.2 风险评估规则 (assess_dry_eye_risk)

```python
# 1. 频率得分 (U型曲线：正常 15-25次/分钟)
if blink_rate < 5:   freq_score = 100      # 眨眼太少 = 干眼
elif blink_rate < 15: freq_score = ...    # 偏低
elif blink_rate <= 25: freq_score = 0     # 正常
elif blink_rate <= 40: freq_score = ...   # 偏高
else: freq_score = 100                    # 眨眼太多 = 干眼

# 2. 时长得分 (U型曲线：正常 100-250ms)
if avg_dur_ms < 50: dur_score = 100        # 太短 = 干眼
elif avg_dur_ms < 100: dur_score = ...     # 偏短
elif avg_dur_ms <= 250: dur_score = 0      # 正常
elif avg_dur_ms <= 500: dur_score = ...     # 偏长
else: dur_score = 100                      # 太长 = 干眼

# 3. 闭合比例得分 (U型曲线：正常 3%-8%)
if closure_ratio < 1.0: closure_score = 100  # 盯着屏幕 = 高风险
elif closure_ratio < 3.0: closure_score = ...
elif closure_ratio <= 8.0: closure_score = 0  # 正常
elif closure_ratio <= 15.0: closure_score = ...  # 偏高
else: closure_score = 100                     # 太高 = 干眼

# 4. 不完全眨眼得分 (线性递增)
if incomplete_ratio_pct < 10: inc_score = 0
else: inc_score = min(100, (incomplete_ratio_pct - 10) * 1.5)

# 5. 长眨眼得分 (线性递增)
if long_blink_ratio_pct < 5: long_score = 0
else: long_score = min(100, (long_blink_ratio_pct - 5) * 5)

# 综合评分
weights = {'freq': 0.15, 'duration': 0.15, 'closure': 0.25, 'incomplete': 0.30, 'long': 0.15}
risk_score = freq*0.15 + dur*0.15 + closure*0.25 + inc*0.30 + long*0.15
```

### 5.3 风险等级划分

```python
if risk_score < 30:   level = "低风险"   # 绿色
elif risk_score < 60: level = "中风险"   # 黄色
else:                  level = "高风险"   # 红色
```
