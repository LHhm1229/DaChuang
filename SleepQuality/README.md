
# 睡眠质量检测系统

本系统旨在通过采集眼睑信号并分析眼部运动（SEM/REM），实现对睡眠质量的实时监测与分期。

## 主要功能

- **实时睡眠分期**：自动识别清醒、浅睡、深睡及快速眼动（REM）阶段。
- **眼部运动分析**：检测慢速眼动（SEM）和快速眼动（REM）事件。
- **睡眠评分系统**：根据睡眠深度和稳定性提供综合评分。
- **多端数据同步**：支持蓝牙及 WebSocket 实时数据传输。
- **可视化仪表盘**：直观展示当前睡眠状态、历史趋势及传感器质量。

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.
